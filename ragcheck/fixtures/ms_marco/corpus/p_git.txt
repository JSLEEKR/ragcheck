Git is a distributed version control system created by Linus Torvalds in
2005. Every developer has a full copy of the repository history locally,
enabling fast branching, merging, and offline work. Core objects include
blobs (file contents), trees (directories), commits (snapshots), and tags.
Common hosting platforms include GitHub, GitLab, and Bitbucket. Popular
workflows are Git Flow, GitHub Flow, and trunk-based development.
